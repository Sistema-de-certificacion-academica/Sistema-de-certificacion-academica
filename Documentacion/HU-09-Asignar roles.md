# Asignar Roles a Usuarios

## Historia de Usuario

**Como** administrador del sistema

**Quiero** asignar el rol correspondiente a cada usuario al momento de su registro o mediante una actualización posterior, seleccionando entre Estudiante, Personal Universitario, Administrador o Cliente Externo

**Para** que el sistema aplique automáticamente los permisos y restricciones de acceso asociados a cada rol, garantizando que cada actor interactúe únicamente con las funcionalidades que le corresponden y previniendo accesos no autorizados
