# Actualizar Información del Perfil de Usuario

## Historia de Usuario

**Como** administrador del sistema

**Quiero** actualizar la información de perfil de un usuario existente, incluyendo datos personales, correo electrónico o rol asignado, validando que los cambios no generen conflicto con otros perfiles registrados

**Para** mantener los datos del sistema vigentes ante cambios institucionales como correcciones de información o reasignación de responsabilidades, sin necesidad de eliminar y recrear el perfil
